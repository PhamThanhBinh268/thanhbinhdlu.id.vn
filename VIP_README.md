# 🎉 VIP PACKAGE - QUICK START

## ⚡ Khởi Động Nhanh

### 1️⃣ Khởi tạo dữ liệu VIP (chỉ 1 lần)
```bash
cd server
node init_vip_packages.js
```

### 2️⃣ Khởi động server
```bash
cd server
npm start
```

### 3️⃣ Truy cập trang VIP
Mở browser: `http://localhost:8080/vip-packages.html`
